Lộ trình học tập

Chọn con đường phù hợp nhất để trở thành developer chuyên nghiệp

Frontend Developer

Xây dựng giao diện web đẹp mắt, responsive và tương tác mượt mà với các framework hiện đại.

Tháng 1-2: Nền tảng

HTML, CSS, JavaScript cơ bản

Tháng 3-4: JavaScript nâng cao

ES6+, DOM, Async/Await, API

Tháng 5-7: Framework

React / Vue.js / Angular

Tháng 8-9: Dự án thực tế

Build portfolio, deploy project

9 tháng 15-30 triệu/tháng Nhu cầu rất cao

Mục tiêu đầu ra

  • Thành thạo HTML5 semantic, CSS3, Flexbox, Grid
  • Master JavaScript ES6+ và ít nhất 1 framework
  • Build responsive, cross-browser websites
  • Có portfolio với 3-5 dự án thực tế

Backend Developer

Xây dựng server, API, database và xử lý logic phía sau để tạo nên hệ thống vững chắc.

Tháng 1-2: Nền tảng & ngôn ngữ

JavaScript/Python, Git, Linux basics

Tháng 3-4: Database & SQL

MySQL/PostgreSQL, MongoDB, ORM

Tháng 5-7: Framework & API

Node.js/Django/Spring Boot, REST API

Tháng 8-10: DevOps & Deploy

Docker, CI/CD, Cloud (AWS/GCP)

10 tháng 18-35 triệu/tháng Rất ổn định

Mục tiêu đầu ra

  • Xây dựng RESTful & GraphQL API
  • Thiết kế database chuyên nghiệp
  • Hiểu authentication, authorization, security
  • Deploy ứng dụng lên cloud

Fullstack Developer

Làm chủ cả Frontend và Backend, xây dựng ứng dụng web hoàn chỉnh từ A đến Z.

Tháng 1-3: Frontend cơ bản

HTML, CSS, JavaScript, React

Tháng 4-6: Backend core

Node.js, Express, Database

Tháng 7-9: Fullstack project

MERN/MEAN Stack, Authentication

Tháng 10-12: Nâng cao

DevOps, Testing, Performance

12 tháng 20-45 triệu/tháng Versatile

Mục tiêu đầu ra

  • Build full-stack application end-to-end
  • Hiểu kiến trúc hệ thống tổng thể
  • Deploy và maintain production app
  • Có 2-3 dự án fullstack hoàn chỉnh

AI / ML Engineer

Xây dựng mô hình AI, phân tích dữ liệu và ứng dụng trí tuệ nhân tạo vào thực tế.

Tháng 1-2: Toán & Python

Linear Algebra, Statistics, Python

Tháng 3-5: Data Science

Pandas, NumPy, Visualization

Tháng 6-9: Machine Learning

Scikit-learn, ML algorithms

Tháng 10-12: Deep Learning

TensorFlow, PyTorch, NLP, CV

12 tháng 25-60 triệu/tháng Hot nhất 2026

Mục tiêu đầu ra

  • Thành thạo Python cho Data Science
  • Build và train ML/DL models
  • Deploy AI models vào production
  • Hiểu NLP, Computer Vision, LLM
